# Задача: "Фермер, вовк, коза, капуста"
## Команда: Костенко Нікіта, Бакалина Анастасія, Шах Ярослав
### Цей репозиторій містить рішення класичної логічної головоломки "Вовк, Коза та Капуста", реалізоване трьома різними мовами програмування: Prolog, C++ та Haskell. Проект демонструє відмінності між логічною, імперативною та функціональною парадигмами при вирішенні задач пошуку шляху.
## Опис задачі:
Фермеру потрібно перевезти через річку вовка, козу та капусту.
- У човні є місце лише для фермера та одного пасажира (або він пливе сам).
- Якщо залишити вовка з козою без нагляду — вовк з'їсть козу.
- Якщо залишити козу з капустою без нагляду — коза з'їсть капусту.
  
**Мета:** Знайти послідовність ходів для безпечного перевезення всіх на інший берег. (Найоптимальніше вважається зробити це за 7 ходів)
